## Technical Parameters

### Lens Focal Lengths
| Length | Use Case |
|--------|----------|
| 14-24mm | Extreme wide, architecture, landscapes |
| 35mm | Street photography, environmental portraits |
| 50mm | Natural perspective, everyday |
| 85mm | Portrait, flattering compression |
| 90mm | Classic portrait, slight telephoto |
| 100mm macro | Close-up, product detail |
| 135mm | Headshots, strong background separation |
| 200mm+ | Sports, wildlife, extreme compression |

### Aperture Effects
| Aperture | Effect |
|----------|--------|
| f/1.4 - f/2 | Extreme bokeh, dreamy, isolated subject |
| f/2.8 - f/4 | Moderate blur, subject separation |
| f/5.6 - f/8 | Sharp throughout, product photography |
| f/11 - f/16 | Deep focus, landscapes |
| f/22+ | Maximum depth, diffraction softening |

### Lighting Terms
| Term | Description |
|------|-------------|
| Rembrandt lighting | Triangle of light on cheek, classic portrait |
| Butterfly lighting | Shadow under nose, glamour |
| Split lighting | Half face lit, dramatic |
| Loop lighting | Small shadow from nose, natural |
| Broad lighting | Lit side toward camera, wider face |
| Short lighting | Lit side away from camera, slimming |
| Rim light | Edge lighting from behind, separation |
| Fill light | Softens shadows, reduces contrast |
| Key light | Main light source |
| Hair light | Separates subject from background |
| Volumetric light | God rays, visible light beams |
| Chiaroscuro | Strong contrast, Renaissance painting style |
| High-key | Bright, low contrast, airy |
| Low-key | Dark, high contrast, moody |
