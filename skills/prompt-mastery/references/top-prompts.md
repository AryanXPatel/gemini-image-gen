## Food Photography Patterns (from top 100)

**High-engagement food prompts share:**
1. Steam/smoke visualization
2. 45-degree overhead or eye-level hero angle
3. Ingredient "explosion" or "floating" compositions
4. Warm color temperature emphasis
5. Texture close-ups (glistening, caramelized, crispy)

**Example template:**
```
[Dish name], high-end culinary photography.
45-degree overhead angle, soft side backlight.
Steam rising from [hot element].
[Key ingredients] arranged with intentional composition.
Shallow depth of field on [focal point].
Warm color temperature, appetizing styling.
No utensils, no text, no distractions.
```

## JSON Structure Pattern (from top-ranked prompts)

Many top prompts use JSON for complex scenes:

```json
{
  "meta": {
    "project_name": "Scene Title",
    "style": "ultra-realistic-photography",
    "aspect_ratio": "2:3"
  },
  "subject": {
    "description": "Main subject details",
    "pose": "Position and gesture",
    "expression": "Emotional state"
  },
  "environment": {
    "location": "Setting description",
    "time": "Time of day",
    "weather": "Atmospheric conditions"
  },
  "technical": {
    "camera": "Camera and lens",
    "lighting": "Light setup",
    "color_grade": "Color treatment"
  },
  "constraints": {
    "avoid": ["list", "of", "things", "to", "exclude"]
  }
}
```

**When to use JSON:** Complex scenes with 5+ distinct elements requiring precise control.
